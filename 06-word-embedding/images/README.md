# README Image Guide

## Application screenshots to capture from the deployed app

Delete the earlier generated mock screenshots and capture these four real application views:

1. `01_application_overview.png`
   - App title, summary, sidebar navigation, and saved-model information.

2. `02_word_explorer_results.png`
   - Use `quality`, `model`, or `electricity`.
   - Include the word index, vector preview, nearest-word results, and local PCA view.

3. `03_sentence_embedding_analysis.png`
   - Use: `quality inspection identifies a manufacturing defect`.
   - Include cleaned text, tokens, indices, matrix shape, and vector preview.

4. `04_semantic_search_results.png`
   - Use: `root cause quality failure`.
   - Include ranked results, cosine similarities, and the download button.

Recommended screenshot guidance:

- Capture only the application content, not unrelated browser tabs.
- Use a desktop width so tables and charts are readable.
- Keep browser zoom at 100%.
- Avoid cutting off titles, metric cards, legends, or table columns.
- Save as PNG using the exact filenames above.
