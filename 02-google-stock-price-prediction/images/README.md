# Streamlit Screenshot Guide

After deployment, capture and save:

```text
01_application_overview.png
02_data_quality_and_price_trend.png
03_next_session_forecast.png
04_model_evaluation_and_baselines.png
05_csv_upload_workflow.png
```

Recommended capture settings:

- desktop browser at approximately 1440 px width;
- no browser bookmarks or personal account information visible;
- bundled sample selected for the main screenshots;
- forecast result and financial disclaimer visible;
- model comparison table readable;
- one separate screenshot showing CSV upload mode.
