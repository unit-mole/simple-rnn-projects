# Streamlit Screenshot Guide

The project README uses the following three deployed-application screenshots:

```text
01_streamlit_application_overview.png
03_model_performance_and_baseline_comparison.png
04_csv_upload_and_download_workflow.png
```

## Screenshot scope

### `01_streamlit_application_overview.png`

Shows the application title, financial disclaimer, data-source selection, forecasting setup, and historical Google closing-price trend.

### `03_model_performance_and_baseline_comparison.png`

Shows the Simple RNN metrics, actual-versus-predicted evaluation, and comparison with the previous-close and five-day mean-return baselines.

### `04_csv_upload_and_download_workflow.png`

Shows the CSV upload mode, cleaned-data preview, generated forecast results, and downloadable output workflow.

## Capture guidance

- Use a desktop browser at a readable width.
- Avoid exposing bookmarks, account information, or unrelated browser content.
- Keep the key application content and chart labels readable.
- Save files as PNG using the exact names above.
- Commit all three screenshots with the project README so GitHub can render them.
