# Application Screenshot Guide

After the public Streamlit deployment is tested, save four screenshots with these exact
filenames:

```text
01_streamlit_application_overview.png
02_single_review_prediction.png
03_batch_sentiment_workflow.png
04_model_performance_and_error_analysis.png
```

## 1. Application overview

Capture the title, project explanation, warning, sidebar modes, input-window information,
and decision threshold.

## 2. Single review prediction

Capture the review input, predicted sentiment, positive probability, confidence,
interpretation, and processed-review details.

## 3. Batch sentiment workflow

Capture the CSV or sample-review preview, sentiment-distribution chart, scored table, and
download button.

## 4. Model performance and error analysis

Capture the main metrics, baseline comparison, at least one evaluation chart, and the
error-analysis table.

Use PNG format, readable desktop width, and avoid unrelated browser or account details.
