# Monorepo Integration

This project is already placed in the final portfolio structure:

```text
simple-rnn-projects/
└── 01-electricity-consumption-forecasting/
```

The repository-level README links to this project, and the GitHub Actions
workflow uses the same numbered folder path.

## Streamlit Entrypoint

```text
01-electricity-consumption-forecasting/app/streamlit_app.py
```

## GitHub Actions Workflow

```text
.github/workflows/electricity-rnn-ci.yml
```

Do not rename the project folder after deployment unless the workflow,
Streamlit entrypoint, and README links are updated at the same time.
