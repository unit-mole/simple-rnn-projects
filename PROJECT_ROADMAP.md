# Simple RNN Portfolio Roadmap

This roadmap tracks the planned development order for the `simple-rnn-projects` portfolio.

| Phase | Project | Primary Learning Goal | Planned Deliverable | Status |
|---:|---|---|---|---|
| 1 | Electricity Consumption Forecasting | Time-series windows, chronological validation, Simple RNN regression | Trained model, evaluation suite, Streamlit forecast app | Portfolio-ready |
| 2 | Google Stock Price Prediction | Financial sequence modeling and leakage-aware backtesting | Price forecasting case study and interactive charting | Planned |
| 3 | IMDb Data Analysis | Tokenization, padded sequences, sentiment classification | Review sentiment classifier and batch-scoring app | Planned |
| 4 | SMS Spam Detection | Compact NLP pipeline and binary sequence classification | Spam probability classifier and message demo | Planned |
| 5 | Text Generation | Many-to-one or many-to-many sequence modeling | Interactive text-generation demonstration | Planned |
| 6 | Word Embedding | Learned representations and semantic similarity | Embedding exploration and nearest-word analysis | Planned |

## Development Standard

A project is marked **Portfolio-ready** only after it contains:

1. A complete recruiter-friendly README.
2. Reproducible preprocessing and training code.
3. A documented dataset and safe sample input.
4. Model artifacts or clear retraining instructions.
5. Correct evaluation and at least one meaningful baseline.
6. Saved charts and tables.
7. Unit or smoke tests.
8. Local-run and deployment instructions.
9. A Streamlit application when an interactive demo adds value.
10. Clearly stated assumptions, limitations, and future improvements.
