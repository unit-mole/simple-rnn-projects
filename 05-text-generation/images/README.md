# README Image Guide

The first four files should be screenshots captured directly from the deployed Streamlit application:

1. `01_application_overview.png`
   - Show the application title, navigation, responsible-use notice, model summary, and project introduction.

2. `02_generation_controls.png`
   - Show the Generate Text interface with the seed prompt, generation length, temperature, top-k, and random seed.

3. `03_generated_text_result.png`
   - Show the generated continuation, qualitative metrics, interpretation, and download control.

4. `04_model_performance_summary.png`
   - Show the Model Performance page with validation metrics, configuration, temperature summary, and baseline table.

The following evaluation charts are generated from committed model artifacts:

5. `05_training_validation_loss.png`
6. `06_training_validation_accuracy.png`
7. `07_temperature_diversity_analysis.png`
8. `08_temperature_repetition_analysis.png`
9. `09_rnn_markov_baseline_comparison.png`

Recommended screenshot settings:

- Use the deployed application rather than the local app.
- Keep the browser zoom around 90–100%.
- Capture only the application area when possible.
- Avoid including browser tabs, personal bookmarks, account information, or desktop content.
- Use PNG format.
- Keep filenames exactly as listed above.
