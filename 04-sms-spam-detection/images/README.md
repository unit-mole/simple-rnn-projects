# Streamlit Screenshot Guide

After deployment, save:

```text
01_streamlit_application_overview.png
02_single_message_prediction.png
03_batch_spam_detection_workflow.png
04_model_performance_and_error_analysis.png
```

Capture the responsible-use warning, main workflows, prediction metrics,
batch summary and download, model comparison, evaluation charts, and error
analysis. Use PNG format and avoid account or personal browser information.
