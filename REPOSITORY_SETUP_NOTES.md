# Repository Setup Notes

The downloadable folder is intentionally named:

```text
simple-rnn-projects
```

Place this folder directly inside your local `GIT Projects` directory so that it appears beside:

```text
ann-deep-learning-projects
simple-rnn-projects
```

The ZIP does **not** contain a `.git` folder. Git creates that hidden folder only after the repository is initialized or cloned. The exact Windows CMD commands can be run in the next step after you review the downloaded structure.

The first project is complete. Projects 02–06 currently contain roadmap README files so that all six project folders appear in the correct order on GitHub from the beginning.
